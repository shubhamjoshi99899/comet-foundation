IT training

COMET offers the following trainings to its students to transform their potential skills into success stories -

  • IT Job profile-based trainings

    • We prepare the students for following IT job profiles. We provide students the Live project development opportunity with exposure to play the following roles –

    ✔  Full stack developer ✔  Data analyst ✔  Web developer
    ✔   Application developer ✔   PMO Member ✔  IT technical support
    ✔  Front-end developer ✔  Dev lead ✔  Software tester – manual
    ✔  Database developer ✔  Test lead ✔  Software tester – automation
    ✔  Project coordinator ✔  Scrum master



  • IT tools and technology-based trainings

    • We rigorously train the students for two years on the following tools and technology –

    Level-1
    ✔  Computer basics ✔  Application of IT ✔  MS Office (MS word, excel PowerPoint).
    ✔  IT common terminology ✔  Motivational session ✔  Group discussion classes
    ✔  Spoken English classes

    Level-2
    ✔  HTML, DHTML, CSS, XML ✔  Writing algorithms ✔  Java framework, core Java, spring
    ✔  51, RDBMS – SQL, MySQL ✔  Java script, React JS, Node JS ✔  Code management (Git/GitHub)
    ✔  C- Programming language ✔  Python ✔  Object oriented programming, C++
    ✔  Creating Process Flow Chart

    Level-3
    ✔  Application development best practices ✔  Cloud computing – AWS ✔  Database development best practices
    ✔  DevOps – tools and life cycle and demos ✔  Website development best practices ✔  AI tools associated with different skills

    Level-4
    ✔  Agile software development ✔  ITIL Framework, ITSM ✔  Waterfall, Scrum, Kanban, Safe Framework.
    ✔  Software testing, types, best practices ✔  Resume development ✔  Cloud – architecture, services list, challenges, and benefits
    ✔  Project management, types & methodologies ✔  Job search techniques ✔  SDLC (tools, phases, team, artefacts, metrics and reports)
